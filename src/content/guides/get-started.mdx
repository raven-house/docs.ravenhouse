# Get Started

This guide will help you set up everything you need to start using Raven House.

## Prerequisites

- A compatible wallet (Azguard or Obsidion)
- A Discord account (if you plan to use Discord integration features)

## Wallet Guide

Raven House seamlessly integrates with [Azguard](https://chromewebstore.google.com/detail/azguard-wallet/pliilpflcmabdiapdeihifihkbdfnbmn?hl=en&authuser=0) and [Obsidion](https://app.obsidion.xyz), the leading wallets in the Aztec ecosystem, ensuring secure and private interactions for all users.


[**How to set up Azguard Wallet →**](/wallets/azguard) 

[**How to set up Obsidion Wallet →**](/wallets/obsidion)


## Connect Your Wallet to Raven House

Once you have set up your preferred wallet:

1. Visit [app.ravenhouse.xyz](https://app.ravenhouse.xyz)
2. Click the **Connect Wallet** button in the top right corner
3. Choose your wallet
4. Complete the connection process
5. Once connected, your wallet address will be displayed in the header
